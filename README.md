# alarm-server

### Setup steps
1. pip3 install pyenv
2. pyenv virtualenv <python-version> <environment_name> : 가상환경 생성
3. pyenv local <environment_name> : 가상환경을 해당 프로젝트에서만 사용하겠다
4. pyenv activate <environment_name> : 가상환경 활성화
5. pip install poetry
6. poetry install
